# test-A-B
A/B тестирование интерфейса интернет-магазинf

##  Описание проекта
Анализ результатов A/B теста, сравнивающего текущий дизайн карточки товара с новой версией.

##  Цели проекта
1. Оценить статистическую значимость разницы в конверсии между группами
2. Проанализировать влияние на другие метрики
3. Определить сегменты пользователей, наиболее отзывчивые к изменениям
4. Сформулировать бизнес-рекомендации по внедрению изменений

##  Гипотеза
**Нулевая гипотеза (H0):** Нет разницы в конверсии между контрольной и тестовой группами
**Альтернативная гипотеза (H1):** Новая версия увеличивает конверсию на 15%

##  Технологический стек
- **База данных:** Microsoft SQL Server
- **Анализ:** Python (Pandas, SciPy, Statsmodels), T-SQL
- **Визуализация:** Matplotlib, Seaborn
- **Статистика:** Z-тест, T-тест, доверительные интервалы, мощность теста

##  Ключевые метрики
1. **Primary Metric:** Конверсия в покупку
2. **Secondary Metrics:**
   - Конверсия в добавление корзины
   - Средний чек
   - Средняя длительность сессии
   - Глубина прокрутки страницы
   - Время до конверсии

##  План анализа

### Этап 1: Подготовка данных
- Создание структуры БД для A/B теста
- Генерация реалистичных тестовых данных
- Проверка корректности разбиения на группы

### Этап 2: Предварительный анализ
- Расчет базовых метрик по группам
- Проверка равномерности распределения
- Анализ воронки конверсий

### Этап 3: Статистический анализ
- Z-тест для разницы пропорций
- Расчет доверительных интервалов
- Определение статистической значимости
- Анализ мощности теста

### Этап 4: Сегментный анализ
- Анализ по типам устройств
- Анализ по браузерам
- Анализ по источникам трафика
